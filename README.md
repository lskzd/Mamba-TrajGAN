# Mamba-TrajGAN
本项目是《DynPrice-MG: Trajectory Privacy Protection Based on Dynamic Pricing with Mamba-GAN》，实现了对按需出行的出租车轨迹的隐私保护，同时确保合成轨迹能够应用于下游任务。
This project is "DynPrice-MG: Trajectory Privacy Protection Based on Dynamic Pricing with Mamba-GAN", which achieves privacy protection for the trajectories of on-demand taxis and ensures that the synthesized trajectories can be applied to downstream tasks.
